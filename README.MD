# 这是一个工业设备的监测和学习程序

## 这里有三个端 上位机电脑 单片机中间件 下位机传感器（设备） 

## 一共实现两大功能：

* 一个功能能够在上位机通过指令对下位机进行请求，获取
传感器的响应内容和其对应的响应协议格式。并将其记录存储在上位机系统中。

* 另一个功能能够将已存储完成的响应内容和其对应的协议格式重新载入到单片机中间件中，
并由中间件仿真下位机传感器（设备）与上位机连接，对上位机系统的收发程序进行检测。

* 上位机和中间件件有一套自定的通信协议进行数据包的发送接收。

## 具体功能细节：
### 上位机
* 上位机实现串口底层收发，及其封装。
* 实现了一套预指令供用户进行相关的控制。
* 存储下位机设备的通信协议及其内容。

### 单片机中间件
* 转发上位机下位机间的请求和响应。
* 预指令系统。
* 接收上位机数据并仿真下位机传感器。
